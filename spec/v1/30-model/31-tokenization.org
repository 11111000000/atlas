* Modes
- ASCII default: lowercase [a-z0-9_]+
- Unicode mode: NFKC + [[:word:]]+, downcased; optional CamelCase split (additive tokens)

* Fields (SHOULD)
- Name (highest weight), signature, doc1, file path.
